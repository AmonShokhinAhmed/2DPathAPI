PathAPI::Path::Path ( const std::vector< Vector2 > &  positions)
private

Constructor building the path from an ordered list of positions.

The passed vector has to have it's position in the orderer we want the path to be travelled in. As the positions can't contain every possible step the user would want to know about, they should only feature points where the path changes direction, as every other point can be interpolated. This is private because only Pathfinder needs to be able to create a Path.

Parameters
[in]positions- the positions the path consists of
