Constructor building the path from an ordered list of positions. 

The passed vector has to have it\textquotesingle{}s position in the orderer we want the path to be travelled in. As the positions can\textquotesingle{}t contain every possible step the user would want to know about, they should only feature points where the path changes direction, as every other point can be interpolated. This is private because only Pathfinder needs to be able to create a \mbox{\hyperlink{class_path_a_p_i_1_1_path}{Path}}.
