\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_path_a_p_i_1_1_shape}{Shape}} class, storing a list of vectors to define a 2D shape. 

This is the most basic way to represent an object. It consists of the a list of \mbox{\hyperlink{class_path_a_p_i_1_1_vector2}{Vector2}} objects (forming triangles), that defines a basic shape. That shape can be altered by manipulating the \mbox{\hyperlink{class_path_a_p_i_1_1_transform}{Transform}} member variable. In addition the Walkable flag can be used to define if the shape should be part of the walkable terrain, when using it to create a \mbox{\hyperlink{class_path_a_p_i_1_1_navmesh}{Navmesh}}
